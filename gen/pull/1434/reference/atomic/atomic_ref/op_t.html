std::atomic_ref::operator T(C++20)(C++26で削除)

この関数はC++26で、CV修飾された型を受け取れるようにするため、型Tへの変換演算子から型value_typeへの変換演算子に変更された。

operator T() const noexcept;

概要

型Tへの暗黙の型変換

戻り値

load()

例外

投げない

例

#include <iostream>
#include <atomic>

int main()
{
  int value = 1;
  std::atomic_ref<int> x{value};

  int n = x;
  std::cout << n << std::endl;
}
