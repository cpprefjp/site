<cstddef>

std::operator^=(C++17)

namespace std {
  constexpr byte& operator^=(byte& l, byte r) noexcept;
}

概要

byte型のオブジェクトに対して、ビット排他的論理和の複合代入をする。

効果

以下の式と等価の効果をもつ：

return l = l ^ r;

例外

投げない

例

#include <cassert>
#include <cstddef>

int main()
{
  std::byte a{0b0000'0011};
  std::byte b{0b0000'0101};

  a ^= b;

  assert(static_cast<unsigned char>(a) == 0b0000'0110);
}
