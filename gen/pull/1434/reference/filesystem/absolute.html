function
<filesystem>

std::filesystem::absolute(C++17)

namespace std::filesystem {
  path absolute(const path& p);                      // (1)
  path absolute(const path& p, std::error_code& ec); // (2)
}

概要

パスを絶対パスに変換する。

効果

オペレーティング・システムに応じて、パスpと同じ場所を指す絶対パスを構築する。

(2)では、エラーが発生した場合、ecにエラー情報を書き込む。エラーが発生しなかった場合は、ecのエラー情報をクリアする。

戻り値

構築された絶対パスを返す。

(2)では、エラーが発生した場合はpath()を返す。

例外

  • (1) : ファイルシステムがエラーを報告する場合がある。それに加えて、指定されたファイルが存在しない場合もエラーである。エラーが発生した場合は、std::filesystem::filesystem_error例外を送出する
  • (2) : 仕様上は未規定だが、パスのメモリ確保で例外が発生する可能性がある

備考

  • エラーが発生しない限り、この関数によって返されたパスrpは、rp.is_absolute() == trueである
  • シンボリックリンクの解決や、セカンダリ・ハードディスクへのクエリが必要な場合には、canonical()関数の使用を検討すること
  • 実装に対して、パスpにファイルが存在することをチェックしないよう強く推奨しており、!exists(p)であってもエラーにはならない
  • POSIXベースシステムでの実装は、単純にcurrent_path()/pとなる。Windowsの場合は、GetFullPathNameW()関数と同じ意味論となる

例

#include <iostream>
#include <filesystem>

namespace fs = std::filesystem;

int main()
{
  // 現在の作業ディレクトリ直下の (存在しない) ファイル"a.txt"の、絶対パスを取得する
  fs::path p = fs::absolute("a.txt");
  std::cout << p << std::endl;
}

出力例

"/home/my_app/a.txt"
