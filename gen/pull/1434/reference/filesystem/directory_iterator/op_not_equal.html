std::filesystem::operator!=(C++17)

namespace std::filesystem {
  bool operator!=(const directory_iterator& a, const directory_iterator& b) noexcept;
}

概要

2つのdirectory_iteratorオブジェクトが非等値かを判定する。

戻り値

2つのdirectory_iteratorオブジェクトが非等値である場合はtrue、そうでなければfalseを返す。

例

#include <cassert>
#include <filesystem>
#include <fstream>

namespace fs = std::filesystem;

int main()
{
  fs::create_directory("dir");
  std::ofstream{"dir/a.txt"};

  fs::directory_iterator a{"dir"};
  fs::directory_iterator end{};
  assert(a != end);

  ++a;
  assert(!(a != end));
}
