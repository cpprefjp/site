<flat_map>

std::flat_map::clear(C++23)

void clear() noexcept; // (1) C++23

概要

コンテナ内の全ての要素を削除する。それぞれのデストラクタが呼ばれ、コンテナから削除される。size() は 0 になる。

計算量

線形時間

例外

投げない

例

#include <iostream>
#include <flat_map>

int main ()
{
  std::flat_map<int, char> fm = {
    {3, 'C'},
    {4, 'D'},
    {1, 'A'},
    {2, 'B'},
  };

  std::cout << fm.size() << std::endl;

  fm.clear();

  std::cout << fm.size() << std::endl;
}

出力

4
0
