function
<fstream>

std::basic_ifstream::close

void close();

概要

ファイルを閉じる

効果

rdbuf()->close()を呼び出す。その戻り値が戻り値がヌルポインタだった場合、setstate(failbit)を呼び出す。

例

#include <iostream>
#include <fstream>

int main()
{
  std::ifstream fs("foo");
  fs.close();
  std::cout << std::boolalpha << fs.is_open() << std::endl;
}

出力

false
