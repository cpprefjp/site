std::generator::iterator::operator++(C++23)

iterator& operator++(); // (1)
void operator++(int);   // (2)

概要

ジェネレータコルーチンを再開し、次の値生成を促す。

事前条件

あるgeneratorオブジェクトxにおいて、coroutine_がアクティブスタック*x.active_に含まれること。

効果

  • (1) : 次と等価 
    x.active_->top().resume()
  • (2) : 次と等価: ++*this

戻り値

  • (1) : *this
