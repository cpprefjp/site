std::list::insert_range(C++23)

template <container-compatible-range<T> R>
iterator insert_range(const_iterator pos, R&& rg); // C++23

概要

Rangeの各要素を任意の位置に挿入する。

テンプレートパラメータ制約

型Tが*ranges::begin(rg)からlistコンテナへのEmplaceConstructibleであること。

効果

Rangergの各要素を、posの直前に挿入する。

戻り値

挿入されたRangergの最初の要素を指すイテレータ。rgが空の場合はpos。

例

#include <iostream>
#include <iterator>
#include <list>

int main()
{
  std::list<int> lst = {1, 2, 3};
  const int a[3] = {4, 5, 6};

  // Rangeを１番目と２番目の要素の間に挿入
  lst.insert_range(std::next(lst.begin()), a);

  for (int i : lst) {
    std::cout << i << " ";
  }
  std::cout << std::endl;
}

出力

1 4 5 6 2 3
