is-layout-left-padded-mapping-of(C++26)

template<class Mapping>
constexpr bool is-layout-left-padded-mapping-of = see below;  // exposition only

概要

is-layout-left-padded-mapping-ofは、layout_left_padded<S>::mappingを判定する説明専用変数テンプレートである。

size_t型の任意の値SとしてMappingがlayout_left_padded<S>::mappingの特殊化であるときに限り、is-layout-left-padded-mapping-of<Mapping>はtrueとなる。
