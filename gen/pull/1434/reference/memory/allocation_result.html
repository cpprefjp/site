std::allocation_result(C++23)

namespace std {
  template <class Pointer, class SizeType = std::size_t>
  struct allocation_result {
    Pointer ptr;
    SizeType count;
  };
}

概要

std::allocation_resultは、allocate_at_least()関数の戻り値型であり、以下の要素をもつ：

変数名 説明
ptr 確保されたメモリの先頭を指すポインタ
count 実際に確保された要素数
