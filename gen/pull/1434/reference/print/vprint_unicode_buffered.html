std::vprint_unicode_buffered(C++26)

namespace std {
  void vprint_unicode_buffered(
         FILE* stream,
         string_view fmt,
         format_args args); // (1) C++26
}

概要

確保した文字列バッファに書式指定の出力を作ってからUnicode出力する。

効果

  • (1) : 以下と等価： 
    string out = vformat(fmt, args);
vprint_unicode("{}", make_format_args(out));
