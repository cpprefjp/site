std::chi_squared_distribution::min(C++11)

result_type min() const;

概要

生成し得る値の下限を取得する。

戻り値

生成し得る値の最小値である0を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::chi_squared_distribution<> dist(1.0);

  double min_val = dist.min();
  std::cout << min_val << std::endl;
}

出力例

0
