<random>

std::extreme_value_distribution::reset(C++11)

void reset();

概要

状態をリセットする。
この関数を呼び出すことで、後続の乱数生成が、前回の乱数生成によって変更されうる状態に依存せずに行うことができる。

効果

内部の状態をリセットする。

戻り値

なし

計算量

定数時間

例

#include <iostream>
#include <random>

int main()
{
  std::random_device seed_gen;
  std::default_random_engine engine(seed_gen());

  std::extreme_value_distribution<> dist(0.0, 1.0);

  for (int i = 0; i < 5; ++i) {
    dist.reset(); // 前回生成までの状態をリセット
    std::cout << dist(engine) << std::endl;
  }
}
