std::ranges::ssize(C++20)

namespace std::ranges {
  inline namespace /*unspecified*/ {
    inline constexpr /*unspecified*/ ssize = /*unspecified*/;
  }
}

概要

Rangeの要素数を符号付き整数型で取得する関数オブジェクト。

効果

部分式Eの型をTとする。このとき、式ranges::ssize(E)の効果は以下の式と等しい。

  1. range_difference_t<T>がptrdiff_tより狭ければ、static_cast<ptrdiff_t>(ranges::size(E))。
  2. それ以外のとき、static_cast<range_difference_t<T>>(ranges::size(E))。

戻り値

Rangeの要素数。

カスタマイゼーションポイント

ranges::sizeをカスタマイズすることで、ranges::ssizeをカスタマイズできる。

例

#include <cassert>
#include <vector>
#include <ranges>

int main()
{
  std::vector<int> v = {1, 2, 3};
  int ar[] = {1, 2, 3};

  // コンテナの要素数を取得。
  std::ptrdiff_t n1 = std::ranges::ssize(v);
  assert(n1 == 3);

  // 生配列の要素数を取得
  std::ptrdiff_t n2 = std::ranges::ssize(ar);
  assert(n2 == 3);
}
