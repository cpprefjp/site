std::ranges::view_interface::size(C++20)

constexpr auto size()
  requires forward_range<D> && sized_sentinel_for<sentinel_t<D>, iterator_t<D>>;                   // (1)

constexpr auto size() const
  requires forward_range<const D> && sized_sentinel_for<sentinel_t<const D>, iterator_t<const D>>; // (2)

概要

Rangeの大きさを取得する。

テンプレートパラメータ制約

view_interface<D>に対して、

  • (1): Dがforward_rangeであり、DのイテレータIと番兵Sがsized_sentinel_forを満たすこと。
  • (2): const Dがforward_rangeであり、const DのイテレータIと番兵Sがsized_sentinel_forを満たすこと。

戻り値

(1)、(2)共に、以下と等価：

ranges::end(derived()) - ranges::begin(derived());
