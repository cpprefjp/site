function
<set>

std::set::rend

reverse_iterator rend();           // (1) C++03
reverse_iterator rend() noexcept;  // (1) C++11

const_reverse_iterator rend() const;           // (2) C++03
const_reverse_iterator rend() const noexcept;  // (2) C++11

概要

set コンテナの先頭要素の前（これは反転シーケンスの末尾にあたる）を指す逆イテレータを取得する。

rend() は begin() と同じ要素を指すわけではなく、その前の要素を指すことに注意。

戻り値

反転シーケンスの終端を指す逆イテレータ。

reverse_iterator と const_reverse_iterator はメンバ型である。set クラステンプレートにおいて、これらは双方向イテレータであり、それぞれ reverse_iterator<iterator>, reverse_iterator<const_iterator> と定義される。

例

#include <iostream>
#include <set>

int main()
{
  std::set<int> c;
  c.insert(5);
  c.insert(2);
  c.insert(4);
  c.insert(1);
  c.insert(0);
  c.insert(0);
  c.insert(9);

  std::set<int>::reverse_iterator i = c.rbegin();
  while (i != c.rend())
    std::cout << *i++ << " ";
  std::cout << std::endl;
}

出力

9 5 4 2 1 0
