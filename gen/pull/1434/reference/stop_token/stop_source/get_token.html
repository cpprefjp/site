<stop_token>

std::stop_source::get_token(C++20)

[[nodiscard]]
stop_token get_token() const noexcept; // (1) C++20
stop_token get_token() const noexcept; // (1) C++26

概要

自身と停止状態を共有するstop_tokenを構築して返す。

戻り値

stop_possible() == false のときは、デフォルト構築したstop_tokenを返す。それ以外の場合は、自身と停止状態を共有するstop_tokenを構築して返す。

例外

投げない。

例

#include <cassert>
#include <stop_token>

int main()
{
  std::stop_source ss1;
  std::stop_source ss2(std::nostopstate);

  std::stop_token st1 = ss1.get_token();
  std::stop_token st2 = ss2.get_token();

  assert(st1.stop_possible() == true);
  assert(st2.stop_possible() == false);
}
