std::basic_string::pop_back(C++11)

void pop_back();           // (1) C++11
constexpr void pop_back(); // (1) C++20

概要

末尾の1要素を削除する。

要件

!empty()

効果

erase(size() - 1, 1)

例外

投げない

例

#include <iostream>
#include <string>

int main()
{
  std::string s = "helloo";

  // 末尾の`o`を1つ削除する
  s.pop_back();

  std::cout << s << std::endl;
}

出力

hello
