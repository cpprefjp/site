<unordered_set>

std::unordered_set::cbegin(C++11)

const_iterator cbegin() const noexcept;

概要

先頭の要素を指す読み取り専用イテレータを取得する。

unordered_set は非順序連想コンテナであるため「先頭」に特に意味はないが、cbegin() で得られたイテレータを cend() まで operator++() でイテレートすることで当該コンテナの要素を漏れなくダブりなく走査することができる。

戻り値

先頭の要素を指す読み取り専用イテレータ

例外

投げない。

計算量

定数

例

#include <iostream>
#include <algorithm>
#include <iterator>
#include <unordered_set>

int main()
{
  std::unordered_set<int> us{ 1, 2, 3, };

  std::copy(us.cbegin(), us.cend(), std::ostream_iterator<int>(std::cout, ", "));
  std::cout << std::endl;
}

出力例

3, 2, 1,

注：unordered_set は非順序連想コンテナであるため、出力順序は無意味であることに注意
