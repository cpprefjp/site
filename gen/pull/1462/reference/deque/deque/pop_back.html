std::deque::pop_back

void pop_back();

概要

末尾要素を削除する。

要件

empty() == falseであること。

戻り値

なし

例外

投げない

計算量

定数時間

例

#include <iostream>
#include <deque>

int main()
{
  std::deque<int> c = {1, 2, 3};

  c.pop_back();

  for (int x : c) {
    std::cout << x << std::endl;
  }
}

1
2
