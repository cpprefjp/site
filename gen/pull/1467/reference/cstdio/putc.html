function
<cstdio>

std::putc

namespace std {
  int putc(int c, FILE* stream);
}

概要

指定されたストリームに1文字出力する。

戻り値

成功したら書き込んだ文字を、失敗すればEOFを返す。

例

#include <cstdio>

int main() {
  std::putc('H', stdout);
  std::putc('e', stdout);
  std::putc('l', stdout);
  std::putc('l', stdout);
  std::putc('o', stdout);
  std::putc('\n', stdout);
}

出力

Hello
