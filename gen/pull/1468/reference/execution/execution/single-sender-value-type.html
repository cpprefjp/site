std::execution::single-sender-value-type(C++26)

template<class Sndr, class Env>
using single-sender-value-type = see below;  // exposition only

概要

single-sender-value-typeは、実行制御ライブラリの仕様定義で用いられる説明専用のエイリアステンプレートである。

型SndrとEnvに対して、single-sender-value-type<Sndr, Env>は下記のエイリアスとなる。

  • value_types_of_t<Sndr, Env, decay_t, type_identity_t>が適格であるならば、その型。
  • そうではなく、value_types_of_t<Sndr, Env, tuple, variant>がvariant<tuple<>>もしくはvariant<>ならば、void型。
  • value_types_of_t<Sndr, Env, decayed-tuple, type_identity_t>`が適格であるならば、その型。
  • そうでなければ、single-sender-value-type<Sndr, Env>は不適格。
