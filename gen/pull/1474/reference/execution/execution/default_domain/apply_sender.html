std::execution::default_domain::apply_sender(C++26)

template<class Tag, sender Sndr, class... Args>
constexpr decltype(auto) apply_sender(Tag, Sndr&& sndr, Args&&... args)
  noexcept(see below);

概要

Senderアルゴリズム適用のデフォルト動作。 execution::apply_senderに対するスタマイゼーションポイントとして機能する。

Senderアルゴリズムタグ型Tagに対して、apply_senderメンバ関数を呼び出す。

テンプレートパラメータ制約

説明用の式eをTag().apply_sender(std::forward<Sndr>(sndr), std::forward<Args>(args)...)としたとき、式eが適格であること。

戻り値

eを返す。

例外

noexcept(e) == trueならば、例外を投げない。 そうでなければ、式eから送出される例外。
