std::execution::run_loop::run-loop-scheduler(C++26)

class run-loop-scheduler;  // exposition only

概要

run-loop-schedulerは、get_schedulerメンバ関数の戻り値型として利用される説明専用のクラスである。

クラス仕様

  • run-loop-scheduler型はschedulerのモデルとなる未規定の型。
  • run-loop-schedulerインスタンスは、同インスタンスを取得したrun_loopインスタンスの生存期間(lifetime)終了まで有効。
  • 2個のrun-loop-schedulerインスタンスは、同一run_loopインスタンスから取得された場合に限って等しい。
  • 説明用の式schの型がrun-loop-schedulerであるとき、式schedule(sch)は型run-loop-senderとなり、式schが潜在的に例外送出しない(not potentially-throwing)ならばそれに従う。
