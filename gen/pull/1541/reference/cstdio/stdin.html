stdin

#define stdin implementation-defined

概要

標準入力を表す*FILE型のオブジェクト。

マクロとして定義されているため、std名前空間には存在しない。

例

#include <cstdio>

int main() {
  int n;
  std::FILE* fp = stdin;
  std::FILE* fp2 = stdout;
  std::fscanf(fp, "%d", &n);
  std::fprintf(fp2, "%d\n", n);
}
