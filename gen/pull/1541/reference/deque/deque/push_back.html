<deque>

std::deque::push_back

void push_back(const T& x); // (1)
void push_back(T&& y);      // (2) C++11

概要

末尾に要素を追加する。

効果

  • (1) : xのコピーを末尾に追加する
  • (2) : 一時オブジェクトxを移動して末尾に追加する

戻り値

なし

計算量

定数時間

備考

操作中に例外が発生した場合、副作用は発生しない。

例

#include <iostream>
#include <deque>
#include <string>

int main()
{
  std::deque<std::string> c;

  // const&バージョン
  std::string s = "hello";
  c.push_back(s);

  // &&バージョン
  c.push_back(std::string("world"));

  for (auto x : c) {
    std::cout << x << std::endl;
  }
}

出力

hello
world
