std::execution::single-sender-value-type(C++26)

template<class Sndr, class... Env>
using single-sender-value-type = see below;  // exposition only

概要

single-sender-value-typeは、実行制御ライブラリの仕様定義で用いられる説明専用のエイリアステンプレートである。

型Sndrと型パックEnvに対して、CSをcompletion_signatures_of_t<Sndr, Env...>とする。CSが不適格またはsizeof...(Env) > 1のとき、single-sender-value-type<Sndr, Env...>は不適格となる。

そうでなければ、single-sender-value-type<Sndr, Env...>は下記のエイリアスとなる。

  • gather-signatures<set_value_t, CS, decay_t, type_identity_t>が適格であるならば、その型。
  • そうではなく、gather-signatures<set_value_t, CS, tuple, variant>がvariant<tuple<>>もしくはvariant<>ならば、void型。
  • そうではなく、gather-signatures<set_value_t, CS, decayed-tuple, type_identity_t>が適格であるならば、その型。
  • そうでなければ、single-sender-value-type<Sndr, Env...>は不適格。
