std::basic_format_string::get(C++23)

constexpr basic_string_view<charT> get() const noexcept; // (1)

概要

書式文字列を取得する。

戻り値

メンバ変数としてbasic_string_view<charT> str;が定義されるとして、

return str;

例

#include <cassert>
#include <format>

template <class... Args>
void f(std::format_string<Args...> fmt, Args&&...) {
  assert(fmt.get() == "Hello {} World");
}

int main() {
  f("Hello {} World", 42);
}
