<ios>

std::nouppercase

namespace std {
  ios_base& nouppercase(ios_base& str);
}

概要

出力時に英小文字を使用することを指示するマニピュレータ。 hexやscientific、hexfloatなどと組み合わせることで効果がある。

効果

str.unsetf(std::ios_base::uppercase)を実行する。

戻り値

実引数のstrオブジェクト。

例

#include <iostream>

int main()
{
  std::cout << std::hex;
  std::cout << std::uppercase << 0xbeef << std::endl;
  std::cout << std::nouppercase << 0xbeef << std::endl;
}

出力

BEEF
beef
