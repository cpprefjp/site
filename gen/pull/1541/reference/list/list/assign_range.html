std::list::assign_range(C++23)

template <container-compatible-range<T> R>
void assign_range(R&& rg); // C++23

概要

Rangeの各要素を再代入する。

事前条件

*this の要素の範囲と Rangerg の要素の範囲が重複していないこと

テンプレートパラメータ制約

型Tが*ranges::begin(rg)からlistコンテナへのEmplaceConstructibleであること。

効果

Rangergの各要素を、再代入する。

戻り値

なし

例

#include <iostream>
#include <list>

int main()
{
  std::list<int> lst = {1, 2, 3};
  const int a[3] = {4, 5, 6};

  // Rangeを再代入
  lst.assign_range(a);

  for (int i : lst) {
    std::cout << i << " ";
  }
  std::cout << std::endl;
}

出力

4 5 6
