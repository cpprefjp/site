<memory>

memory

<memory>ヘッダでは、メモリアロケータ、未初期化領域に関する関数群、スマートポインタ、アライメントに関するユーティリティ関数といった、メモリを扱うための機能を定義する。

このヘッダでは、以下の標準ヘッダをインクルードする：

  • <compare> (C++20)
