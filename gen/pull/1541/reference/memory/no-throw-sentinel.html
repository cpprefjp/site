std::no-throw-sentinel(C++20)

namespace std {
  template <class S, class I>
  concept no-throw-sentinel = sentinel_for<S, I>;
}

概要

no-throw-sentinelは、以下の操作で例外を投げない、型Sがイテレータ型Iの番兵（sentinel）型であることを表す説明用コンセプトである。
