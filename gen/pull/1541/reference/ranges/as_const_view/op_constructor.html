std::ranges::as_const_view::コンストラクタ(C++23)

as_const_view() requires default_initializable<V> = default;    // (1)

constexpr explicit as_const_view(V base);                       // (2)

概要

as_const_viewオブジェクトを構築する。

効果

入力view（V）のオブジェクトをbase_というメンバに保持するとして

  • (1) : base_をデフォルト構築する
  • (2) : base_をstd::move(base)で初期化する

例

#include <ranges>
#include <vector>

int main() {
  using std::ranges::as_const_view;

  std::vector<int> vec = {1, 2, 3, 4, 5};

  // (1) デフォルト構築
  as_const_view<std::views::all_t<std::vector<int>>> v1{};

  // (2) viewを入力して構築
  as_const_view v2{vec};
}
