概要

chunk_by_viewはRangeを指定した二項述語が偽となる場所で分割したviewのRange。

  • (1): chunk_by_viewのクラス定義
  • (2): chunk_by_viewを生成するカスタマイゼーションポイントオブジェクト

例

#include <ranges>
#include <vector>
#include <functional>
#include <print>

int main() {
  std::vector v = {1, 2, 2, 3, 0, 4, 5, 2};
  // 列が転倒する場所で分割
  std::println("{}", v | std::views::chunk_by(std::ranges::less_equal{}));
}

出力

[[1, 2, 2, 3], [0, 4, 5], [2]]
