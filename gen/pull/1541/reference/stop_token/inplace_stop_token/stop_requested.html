std::inplace_stop_token::stop_requested(C++26)

bool stop_requested() const noexcept;

概要

停止要求が作成されたかどうかを返す。

効果

自身がinplace_stop_sourceを参照し、かつstop_requestedがtrueの場合に限って、trueを返す。 そうでなければ、falseを返す。

例外

投げない
