std::basic_string_view::swap(C++17)

constexpr void swap(basic_string_view& s) noexcept;

概要

他のbasic_string_viewオブジェクトとデータを入れ替える。

効果

thisとsが保持しているデータを入れ替える。

例外

投げない

備考

basic_string_viewクラスに対して、非メンバ関数版の専用swap()は定義されない。そのような用途には、<utility>ヘッダで定義される汎用版のstd::swap()関数を使用すること。

例

#include <iostream>
#include <string_view>

int main()
{
  std::string_view a = "AAA";
  std::string_view b = "BBB";

  a.swap(b);

  std::cout << a << std::endl;
  std::cout << b << std::endl;
}

出力

BBB
AAA
