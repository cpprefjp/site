std::variant::デストラクタ(C++17)

~variant();            // C++17
constexpr ~variant();  // C++23

概要

variantオブジェクトを破棄する

効果

valueless_by_exception()がfalseである場合、保持している値を破棄する。

備考

  • Types...の全ての型Tiについてis_trivially_destructible_v<Ti>がtrueである場合、この関数はトリビアルとなる
    • つまり、全ての候補型がデストラクタを呼び出す必要のない型である場合、valueless_by_exception()の状態に関わらず何もする必要がない
