std::formatter(C++23)

namespace std {
  template <class charT>
  class formatter<thread::id, charT>;
}

概要

thread::idクラスに対するstd::formatterクラステンプレートの特殊化。

フォーマットフラグとしては、以下を使用できる：

[[fill] [align] [width]]

例

#include <print>
#include <thread>

int main()
{
  std::println("{}", std::this_thread::get_id());
  std::println("{: >30}", std::this_thread::get_id());
}
