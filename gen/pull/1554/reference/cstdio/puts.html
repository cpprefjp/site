function
<cstdio>

std::puts

namespace std {
  int puts(const char* str);
}

概要

引数に渡されたヌル終端文字列を標準出力に出力する。

また、末尾には自動で改行がつく。

戻り値

成功すればEOF以外の値を返し、失敗すればEOFを返す。

例

#include <cstdio>

int main() {
  std::puts("Hello, World!");
}

出力

Hello, World!
