std::showbase

namespace std {
  ios_base& showbase(ios_base& str);
}

概要

整数出力時に基数を表すプレフィックスを付与することを指示するマニピュレータ。 hexやoctと組み合わせると、それぞれ0xおよび0が頭に出力される。 ただし、0の場合を除く。

money_putなど、その他の出力でも効果がある場合がある。

効果

str.setf(std::ios_base::showbase)を実行する。

戻り値

実引数のstrオブジェクト。

例

#include <iostream>

int main()
{
  std::cout << std::hex;
  std::cout << 15 << ' ' << 0 << std::endl;
  std::cout << std::showbase << 15 << ' ' << 0 << std::endl;

  std::cout << std::noshowbase;

  std::cout << std::oct;
  std::cout << 15 << ' ' << 0 << std::endl;
  std::cout << std::showbase << 15 << ' ' << 0 << std::endl;
}

出力

f 0
0xf 0
17 0
017 0
