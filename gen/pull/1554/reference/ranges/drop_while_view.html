std::ranges::drop_while_view(C++20)

namespace std::ranges {
  template<view V, class Pred>
    requires input_range<V> && is_object_v<Pred> && indirect_unary_predicate<const Pred, iterator_t<V>>
  class drop_while_view : public view_interface<drop_while_view<V, Pred>> { …… }; // (1)

  namespace views {
    inline constexpr /*unspecified*/ drop_while = /*unspecified*/;     // (2)
  }
}

概要

  • (1): 元のRangeの先頭から指定した条件を連続して満たす範囲を除外するview
  • (2): drop_while_viewを生成するRangeアダプタオブジェクト

Rangeコンセプト

borrowed sized output input forward bidirectional random_access contiguous common viewable view
(1) (1) (1) (1) (1) (1) (1) ○ ○
  • (1): Vに従う

テンプレートパラメータ制約

  • view<V>
  • input_range<V>
  • is_object_v<Pred>
  • indirect_unary_predicate<Pred, iterator_t<V>>

効果

  • (2): 式views::drop_while(E, F)の効果はdrop_while_view(E, F)と等しい
