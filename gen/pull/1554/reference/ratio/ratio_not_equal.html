std::ratio_not_equal(C++11)

namespace std {
  template <class R1, class R2>
  struct ratio_not_equal;

  template <class R1, class R2>
  inline constexpr bool ratio_not_equal_v
    = ratio_not_equal<R1, R2>::value;     // C++17
}

概要

ratio_not_equalは、2つのratioが等値でないかを判定するクラステンプレートである。

効果

ratio_not_equalは、ratio_equal<R1, R2>::value == falseであればtrue_typeから派生し、そうでなければfalse_typeから派生する。

例

#include <ratio>

int main()
{
  using r1 = std::ratio<2, 5>;
  using r2 = std::ratio<3, 5>;

  static_assert(std::ratio_not_equal<r1, r2>::value == true, "r1 != r2");
}
