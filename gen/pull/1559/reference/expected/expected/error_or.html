std::expected::error_or(C++23)

template<class G = E> constexpr E error_or(G&& e) const &; // (1)
template<class G = E> constexpr E error_or(G&& e) &&;      // (2)

概要

エラー値もしくは指定された値を取得する。

適格要件

  • (1) : is_copy_constructible_v<E> == true && is_convertible_v<G, E> == true
  • (2) : is_move_constructible_v<E> == true && is_convertible_v<G, E> == true

戻り値

  • (1) : has_value() ? std::forward<G>(e) : error()
  • (2) : has_value() ? std::forward<G>(e) : std::move(error())

例

#include <expected>
#include <iostream>
#include <string>

int main()
{
  std::expected<int, std::string> x = 42;
  std::cout << x.error_or("-") << std::endl;

  std::expected<int, std::string> y = std::unexpected{"ERR"};
  std::cout << y.error_or("-") << std::endl;
}

出力

-
ERR
