std::extreme_value_distribution::max(C++11)

result_type max() const;

概要

生成し得る値の上限を取得する。

戻り値

値の範囲の上限を返す。

例

#include <iostream>
#include <random>

int main()
{
  std::extreme_value_distribution<> dist(0.0, 1.0);

  double max_value = dist.max();
  std::cout << max_value << std::endl;
}

出力例

1.79769e+308
