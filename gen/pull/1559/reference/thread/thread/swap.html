<thread>

std::thread::swap(C++11)

void swap(thread& x) noexcept;

概要

別のthreadと交換する。

効果

*thisとxを入れ替える。

例外

送出しない。

例

#include <thread>

int main()
{
  std::thread t1([]{ /*...*/ });
  std::thread t2;

  t1.swap(t2);

  t2.join();

  return 0;
}
