function
<valarray>

std::valarray::size

std::size_t size() const;

概要

要素数を取得する。

戻り値

valarrayオブジェクトに含まれる要素数を返す。

計算量

定数時間

例

#include <iostream>
#include <valarray>

int main()
{
  const std::valarray<int> va = {1, 2, 3};

  std::size_t size = va.size();
  std::cout << size << std::endl;
}

出力

3
