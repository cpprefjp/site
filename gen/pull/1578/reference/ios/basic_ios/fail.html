<ios>

std::basic_ios::fail

bool fail() const;

概要

現在の状態値のうちfailbitとbadbitを判定する。

戻り値

failbitとbadbitのいずれかまたは両方が設定されていればtrue、さもなくばfalse。

備考

もし、本当にfailbitのみの判定を行う必要があれば、rdstate()を使って(rdstate() & failbit) != 0などと記述すればよい。

実装例

bool fail() const {
  return (rdstate() & (failbit | badbit)) != 0;
}
