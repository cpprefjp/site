std::mdspan::mapping(C++23)

constexpr const mapping_type& mapping() const noexcept;

概要

レイアウトマッピングmap_を取得する。

map_

例外

投げない

例

#include <cassert>
#include <mdspan>

int main()
{
  int arr[] = {1, 2, 3, 4, 5, 6};
  // 静的要素数 2x3 の2次元配列ビュー
  std::mdspan<int, std::extents<size_t, 2, 3>> mat{arr};

  auto map = mat.mapping();
  assert(map.required_span_size() == 6);
}
