<initializer_list>

std::initializer_list::data(C++26)

constexpr const E* data() const noexcept; // (1) C++26

概要

配列の先頭へのポインタを取得する。

戻り値

配列の先頭要素を指すポインタを返す。

配列の要素数が空である場合、begin()と同じ未規定のポインタ値を返す。

例外

投げない

例

#include <iostream>
#include <initializer_list>

int main()
{
  std::initializer_list<int> init = {1, 2, 3};

  const int* p = init.data();

  std::cout << *p << std::endl;
}

出力

1
