std::basic_ostream::put

basic_ostream<CharT, Traits>& put(char_type c);

概要

（非書式化出力関数）ストリームへ文字を出力する。

効果

  1. sentry オブジェクトを構築する。sentry オブジェクトが失敗を示した場合、何もしない。
  2. c をストリームバッファへ出力する。
  3. 出力できなかったら、setstate(badbit) を呼び出す。

戻り値

*this

例

#include <iostream>

int main() {
  std::cout.put('1');
  std::cout.put('\n');
}

出力

1
