<span>

std::span::cbegin(C++23)

constexpr const_iterator cbegin() const noexcept;

概要

先頭要素を指す読み取り専用イテレータを取得する。

戻り値

return begin();

例外

投げない

例

#include <iostream>
#include <span>
#include <vector>

int main() {
  std::vector<int> v = {1, 2, 3, 4, 5};
  std::span<int, 5> sp{v};

  auto cit = sp.cbegin();

  std::cout << *cit << '\n';

  // これはできない
  // *cit = 0;
}

1
