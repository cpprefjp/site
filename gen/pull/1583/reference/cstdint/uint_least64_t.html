<cstdint>

std::uint_least64_t(C++11)

namespace std {
  using uint_least64_t = unsigned-integer-type;
}

概要

64ビット以上の、最も小さい符号なし整数型。

uint64_t型が環境によっては定義されないため、そのような状況でこの型を使用する。
