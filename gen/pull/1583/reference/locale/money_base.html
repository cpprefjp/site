std::money_base

namespace std {
  class money_base {
  public:
    enum part { none, space, symbol, sign, value };
    struct pattern { char field[4]; };
  };
}

概要

(ここに、クラスの概要を記載する)

メンバ型

名前 説明
part 金額のフォーマットを表現するための列挙型
pattern 金額のフォーマット型

part列挙型

名前 説明
none フォーマットのない文字を解析するための、省略可能な空白文字
space フォーマットが必要とされる文字を解析するための、省略可能な空白文字
symbol 通貨記号(￥、$など)
sign 金額の正負を表す記号
value 金額の値

patternクラス

名前 説明
char field[4]; charに変換されたpart列挙値の配列
