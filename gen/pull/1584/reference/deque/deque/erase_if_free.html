std::erase_if(C++20)

namespace std {
  template <class T, class Allocator, class Predicate>
  typename deque<T, Allocator>::size_type
    erase_if(deque<T, Allocator>& c, Predicate pred); // (1) C++20
  template <class T, class Allocator, class Predicate>
  constexpr typename deque<T, Allocator>::size_type
    erase_if(deque<T, Allocator>& c, Predicate pred); // (1) C++26
}

概要

指定した条件に合致する要素とその分の領域を、コンテナから削除する。

効果

以下と等価：

auto it = remove_if(c.begin(), c.end(), pred);
auto r = distance(it, c.end());
c.erase(it, c.end());
return r;

戻り値

削除した要素数を返す。

例

#include <iostream>
#include <deque>

int main()
{
  std::deque<int> deq = {3, 1, 4, 5, 2};

  // コンテナdeqから、偶数をすべて削除する
  std::erase_if(deq, [](int x) { return x % 2 == 0; });

  for (int x : deq) {
    std::cout << x << std::endl;
  }
}

出力

3
1
5
