function
<deque>

std::deque::front

reference front();           // (1) C++03
constexpr reference front(); // (1) C++26

const_reference front() const;           // (2) C++03
constexpr const_reference front() const; // (2) C++26

概要

先頭要素への参照を取得する

戻り値

先頭要素への参照

計算量

定数時間

備考

コンテナが空の状態でこの関数が呼ばれた場合、動作は未規定。

例

#include <iostream>
#include <deque>

int main()
{
  std::deque<int> c = {1, 2, 3};

  // 先頭要素を取得する
  int& x = c.front();
  std::cout << x << std::endl;
}

出力

1
