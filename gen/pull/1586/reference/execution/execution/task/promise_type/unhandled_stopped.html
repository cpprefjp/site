std::execution::task::promise_type::unhandled_stopped(C++26)

coroutine_handle<> unhandled_stopped() noexcept;

概要

タスクコルーチンの停止完了動作を制御する。

効果

set_stopped(std::move(RCVR(*this)))を呼び出すことで、STATE(*this)に関連付けられた非同期操作を完了する。

戻り値

noop_coroutine()

例外

投げない
