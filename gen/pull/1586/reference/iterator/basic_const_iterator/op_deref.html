<iterator>

std::basic_const_iterator::operator*(C++23)

constexpr reference operator*() const;

概要

イテレータを間接参照する。

戻り値型referenceは、iter_const_reference_t<Iterator>のエイリアス。

効果

ラップしているイテレータをcurrent_メンバ変数に保持するとして、以下と等価

return static_cast<reference>(*current_);

例

#include <iostream>
#include <iterator>
#include <vector>


int main() {
  std::vector vec = {1, 2, 3, 4, 5};

  std::basic_const_iterator cit = vec.begin();

  std::cout << *cit << '\n';

  // 要素を変更できない
  //*cit = 0;
}
