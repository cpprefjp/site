std::execution::get_delegation_scheduler(C++26)

namespace std::execution {
  struct get_delegation_scheduler_t { unspecified };
  inline constexpr get_delegation_scheduler_t get_delegation_scheduler{};
}

概要

get_delegation_schedulerは、クエリ可能オブジェクトから委任Schedulerを取得するクエリオブジェクトである。

コア定数式forwarding_query(get_delegation_scheduler)はtrue値を返す。

効果

呼び出し式get_delegation_scheduler(env)は、MANDATE-NOTHROW(AS-CONST(env).query(get_delegation_scheduler))と等価である。

上記の式が適格なとき、その型はschedulerを満たすこと。

例外

投げない

カスタマイゼーションポイント

const修飾クエリ可能オブジェクトcenvに対して式cenv.query(get_delegation_scheduler)が呼び出される。 このとき、noexcept(cenv.query(get_delegation_scheduler)) == trueであること。
