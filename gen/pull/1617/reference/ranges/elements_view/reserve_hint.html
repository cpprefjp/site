std::ranges::elements_view::reserve_hint(C++26)

constexpr auto reserve_hint()
  requires approximately_sized_range<V>;       // (1) C++26

constexpr auto reserve_hint() const
  requires approximately_sized_range<const V>; // (2) C++26

概要

要素数の近似値を取得する。

テンプレートパラメータ制約

  • (1) : 型Vがapproximately_sized_rangeであること
  • (2) : 型const Vがapproximately_sized_rangeであること

戻り値

元のRangebase_に対して、以下を返す：

return ranges::reserve_hint(base_);
