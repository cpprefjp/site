жҰӮиҰҒ

get_completion_scheduler<completion-tag>гҒҜгҖҒSenderгҒ®еұһжҖ§гҒӢгӮүжҢҮе®ҡе®ҢдәҶгӮҝгӮ°гҒ«й–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҹе®ҢдәҶSchedulerгӮ’еҸ–еҫ—гҒҷгӮӢгӮҜгӮЁгғӘгӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ§гҒӮгӮӢгҖӮ

гӮігӮўе®ҡж•°ејҸforwarding_query(get_completion_scheduler<completion-tag>)гҒҜtrueеҖӨгӮ’иҝ”гҒҷгҖӮ

е®ҢдәҶScheduler

е®ҢдәҶScheduler(completion scheduler)гҒҜгҖҒйқһеҗҢжңҹж“ҚдҪңгҒ®е®ҢдәҶж“ҚдҪңгӮ’е®ҹиЎҢгҒҷгӮӢгҒҹгӮҒгҒ®е®ҹиЎҢгғӘгӮҪгғјгӮ№пјҲдҫӢпјҡCPUгӮ№гғ¬гғғгғүпјүгҒЁй–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҹSchedulerгҒ§гҒӮгӮӢгҖӮ

йқһеҗҢжңҹж“ҚдҪңгҒ®е®ҢдәҶж“ҚдҪңгҒҜгҖҒдёӢиЁҳгҒ„гҒҡгӮҢгҒӢгҒ®е®ҢдәҶй–ўж•°е‘јгҒіеҮәгҒ—гҒҢи©ІеҪ“гҒҷгӮӢгҖӮ

  • еҖӨе®ҢдәҶй–ўж•° execution::set_value
  • гӮЁгғ©гғје®ҢдәҶй–ўж•° execution::set_error
  • еҒңжӯўе®ҢдәҶй–ўж•° execution::set_stopped

еҠ№жһң

completion-fnгӮ’е®ҢдәҶй–ўж•°гҖҒcompletion-tagгӮ’е®ҢдәҶй–ўж•°гҒ«й–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҹе®ҢдәҶгӮҝгӮ°гҖҒargsгҒЁenvsгӮ’йғЁеҲҶејҸгҒ®гғ‘гғғгӮҜгҖҒејҸsndrгӮ’sender<decltype((sndr))>гҒҢtrueгҒӢгҒӨget_completion_scheduler<completion-tag>(get_env(sndr), envs...)гҒЁгҒӘгӮӢејҸгҖҒschгӮ’SchedulerгҒЁгҒҷгӮӢгҖӮ

ејҸsch1гҒЁгғ‘гғғгӮҜenvsгҒ«еҜҫгҒ—гҒҰгҖҒејҸsch2гӮ’TRY-QUERY(sch, get_completion_scheduler<completion-tag>, envs...)гҒЁгҒҷгӮӢгҖӮsch2гҒҢдёҚйҒ©ж јгӮӮгҒ—гҒҸгҒҜsch1гҒЁsch2гҒҢеҗҢдёҖеһӢгҒӢгҒӨзӯүгҒ—гҒ„гҒЁгҒҚгҖҒејҸRECURSE-QUERY(sch1, envs...)гӮ’sch1гҒЁзӯүдҫЎгҒӘејҸгҒЁгҒҷгӮӢгҖӮгҒқгҒҶгҒ§гҒӘгҒ‘гӮҢгҒ°гҖҒејҸRECURSE-QUERY(sch2, envs...)гҒЁйҖҸйҒҺгҒӘејҸгҒЁгҒҷгӮӢгҖӮ

ејҸqгҒЁгғ‘гғғгӮҜenvsгҒ«еҜҫгҒ—гҒҰгҖҒе®ҢдәҶгӮҝгӮ°completion-tagгҒҢset_value_tпјҸset_error_tпјҸ set_stopped_tгҒ®гҒ„гҒҡгӮҢгҒ§гӮӮгҒӘгҒ‘гӮҢгҒ°гҖҒе‘јгҒіеҮәгҒ—ејҸget_completion_scheduler<completion-tag>(q, envs...)гҒҜдёҚйҒ©ж јгҒЁгҒӘгӮӢгҖӮ гҒқгҒҶгҒ§гҒӘгҒ‘гӮҢгҒ°гҖҒе‘јгҒіеҮәгҒ—ејҸгҒҜдёӢиЁҳгҒЁзӯүдҫЎгҖӮ

  • envs...гҒҢ1еӣһгҒ гҒ‘и©•дҫЎгҒ•гӮҢгӮӢгҒ“гҒЁгӮ’йҷӨгҒ„гҒҰгҖҒйҒ©ж јгҒӘгӮүгҒ°дёӢиЁҳгҒ®ејҸ

    MANDATE-NOTHROW(RECURSE-QUERY(
  TRY-QUERY(q, get_completion_scheduler<completion-tag>, envs...), envs...))

  • гҒқгҒҶгҒ§гҒҜгҒӘгҒҸгҖҒenvs...гҒҢ1еӣһгҒ гҒ‘и©•дҫЎгҒ•гӮҢгӮӢгҒ“гҒЁгӮ’йҷӨгҒ„гҒҰгҖҒеһӢqгҒҢschedeulerгӮ’жәҖгҒҹгҒ—гҒӢгҒӨenvsгҒҢз©әгҒ®гғ‘гғғгӮҜгҒ§гҒӘгҒ„гҒЁгҒҚгҖҒauto(q)

  • гҒқгҒҶгҒ§гҒӘгҒ‘гӮҢгҒ°гҖҒе‘јгҒіеҮәгҒ—ејҸget_completion_scheduler<completion-tag>(q, envs...)гҒҜдёҚйҒ©ж јгҒЁгҒӘгӮӢгҖӮ

get_completion_scheduler<completion-tag>(q, envs...)гҒҢйҒ©ж јгҒӘгӮүгҒ°гҖҒгҒқгҒ®еһӢгҒҜSchedulerгӮ’жәҖгҒҹгҒҷгҒ№гҒҚгҖӮ

гӮ«гӮ№гӮҝгғһгӮӨгӮјгғјгӮ·гғ§гғігғқгӮӨгғігғҲ

constдҝ®йЈҫгӮҜгӮЁгғӘеҸҜиғҪгӮӘгғ–гӮёгӮ§гӮҜгғҲcqгҒ«еҜҫгҒ—гҒҰејҸcq.query(get_completion_scheduler<completion-tag>)гҒҢе‘јгҒіеҮәгҒ•гӮҢгӮӢгҖӮ гҒ“гҒ®гҒЁгҒҚгҖҒnoexcept(cq.query(get_completion_scheduler<completion-tag>)) == trueгҒ§гҒӮгӮӢгҒ“гҒЁгҖӮ

еһӢTagгҖҒејҸsndrгҖҒгғ‘гғғгӮҜenvsгҒ«еҜҫгҒ—гҒҰгҖҒCSгӮ’completion_signatures_of_t<decay_t<decltype((sndr))>, decltype((envs))...>гҒЁгҒҷгӮӢгҖӮget_completion_scheduler<Tag>(get_env(sndr), envs...)гҒЁCSгҒҢдёЎиҖ…гҒЁгӮӮйҒ©ж јгҖҒгҒӢгҒӨCS().count-of(Tag()) == 0гҒҢtrueгҒ®гҒЁгҒҚгҖҒгғ—гғӯгӮ°гғ©гғ гҒҜдёҚйҒ©ж јгҒЁгҒӘгӮӢгҖӮ

sndrгҒЁReceiverrcvrгӮ’жҺҘз¶ҡгҒ—гҒҰдҪңжҲҗгҒ•гӮҢгҒҹйқһеҗҢжңҹж“ҚдҪңгҒҢе®ҢдәҶй–ўж•°completion-fn(rcvr, args...)гӮ’и©•дҫЎгҒҷгӮӢгҒЁгҒҚгҖҒгҒқгҒ®и©•дҫЎгҒҢschгҒЁй–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҹе®ҹиЎҢгғӘгӮҪгғјгӮ№гҒ®е®ҹиЎҢгӮЁгғјгӮёгӮ§гғігғҲдёҠгҒ§иЎҢгӮҸгӮҢгҒӘгҒ„е ҙеҗҲгҖҒеӢ•дҪңгҒҜжңӘе®ҡзҫ©гҒЁгҒӘгӮӢгҖӮ

#include <execution>
namespace ex = std::execution;

int main()
{
  ex::run_loop loop;
  ex::scheduler auto loop_sch = loop.get_scheduler();

  // schedule(loop_sch)гҒ®е®ҢдәҶSchedulerгҒҜloop_schгҒ«зӯүгҒ—гҒ„
  ex::sender auto snd0 = ex::schedule(loop_sch);
  auto sch0 = ex::get_completion_scheduler<ex::set_value_t>(ex::get_env(snd0));
  assert(sch0 == loop_sch);

  // е®ҢдәҶSchedulerгҒҜжҺҘз¶ҡгҒ•гӮҢгҒҹSenderгҒёгҒЁеј•гҒҚз¶ҷгҒҢгӮҢгӮӢ
  ex::sender auto snd1 = snd0 | ex::then([]{ return 42; });
  auto sch1 = ex::get_completion_scheduler<ex::set_value_t>(ex::get_env(snd1));
  assert(sch1 == loop_sch);

#if 0
  // just SenderгҒҜе®ҢдәҶSchedulerгӮ’жҢҒгҒҹгҒӘгҒ„
  ex::sender auto snd2 = ex::just(42);
  auto sch2 = ex::get_completion_scheduler<ex::set_value_t>(ex::get_env(snd2));
#endif
}
