## 概要
変換が成功したかを判定する。


## 戻り値
```cpp
return ec == errc{};
```
* errc[link /reference/system_error/errc.md]


## 例
```cpp example
#include <iostream>
#include <charconv>
#include <string_view>

int main()
{
  char out[8]{};
  int value = 123;

  if (auto result = std::to_chars(std::begin(out), std::end(out), value)) {
    std::cout << std::string_view(out, result.ptr - out) << std::endl;
  }
  else {
    std::cout << "conversion failed" << std::endl;
  }
}
```

### 出力
```
123
```
