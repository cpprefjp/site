```cpp
namespace std {
  template<class T>
  concept class-type = decays-to<T, T> && is_class_v<T>;  // exposition only
}
```
* decays-to[link decays-to.md]

## 概要
`class-type`は、型`T`がクラス型であることを表す説明専用のコンセプトである。
