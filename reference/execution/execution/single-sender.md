# single-sender
* execution[meta header]

```cpp
template<class Sndr, class... Env>
concept single-sender = sender_in<Sndr, Env...> &&
  requires {
    typename single-sender-value-type<Sndr, Env...>;
  };
```

## 概要
`single-sender`は、実行制御ライブラリの仕様定義で用いられる説明専用のエイリアステンプレートである。
