# size

```cpp
std::size_t size() const noexcept;           // C++11
constexpr std::size_t size() const noexcept; // C++14
```

## 概要
要素数を取得する。


## 戻り値
`*this`に含まれる、配列の要素数を返す。


## 例外
投げない


## 例
```cpp example
#include <iostream>
#include <initializer_list>

int main()
{
  std::initializer_list<int> init = {1, 2, 3};

  std::size_t n = init.size();
  std::cout << n << std::endl;
}
```
