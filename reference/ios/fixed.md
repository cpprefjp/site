# fixed
* ios[meta header]
* std[meta namespace]
* function[meta id-type]

```cpp
namespace std {
  ios_base& fixed(ios_base& str);
}
```

## 概要
浮動小数点数を固定小数点表記で入出力することを指示するマニピュレータ。

## 効果
`str.setf(ios_base::fixed, ios_base::floatfield)`を実行する。

## 戻り値
実引数のstrオブジェクト。
