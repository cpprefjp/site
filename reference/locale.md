# locale
* locale[meta header]

`<locale>`ヘッダでは、ローカライゼーション（地域化）に関するクラス・関数を定義する。このヘッダに含まれるクラス・関数は`std`名前空間で定義される。


## ロケール

| 名前 | 説明 | 対応バージョン |
|--------------------------------|-----------------------------|-------|
| [`locale`](locale/locale.md) | ロケール(class) | |
| `use_facet` | ロケールが持つファセットの取得(function) | |
| `has_facet` | ロケールがファセットを持っているか判定(function) | |
