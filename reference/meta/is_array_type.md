# is_array_type
* meta[meta header]
* std::meta[meta namespace]
* function[meta id-type]
* cpp26[meta cpp]

```cpp
namespace std::meta {
  consteval bool is_array_type(info type);
}
```
* info[link info.md]

## 概要
配列型であるかを判定する。


## 戻り値
`type`が配列型を表す場合に`true`を返す。


## 例外
`type`が型を表さない場合、[`std::meta::exception`](exception.md)例外を送出する。


## 例
```cpp example
#include <meta>
int main() {
  static_assert(std::meta::is_array_type(^^int[3]));
  static_assert(!std::meta::is_array_type(^^int));
}
```
