## 概要
元となる乱数生成器を取得する。


## 戻り値
メンバ変数として保持している、元となる乱数生成器への`const`参照を返す。


## 例
```cpp example
#include <iostream>
#include <random>
#include <cstdint>

int main()
{
  std::independent_bits_engine<std::mt19937, 64, std::uint64_t> engine;

  // 元となる乱数生成器を取得
  const std::mt19937& base_engine = engine.base();
}
```
