## 概要

## 戻り値
最小値である`0`を返す。


## 計算量

## 例
```cpp example
#include <iostream>
#include <random>

int main()
{
  using engine_type = std::independent_bits_engine<std::mt19937, 64, std::uint64_t>;

  std::uint64_t min_value = engine_type::min();
  std::cout << min_value << std::endl;
}
```
