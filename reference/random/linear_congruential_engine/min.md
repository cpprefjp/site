```cpp
static constexpr result_type min() { return c == 0u ? 1u: 0u; }
```

## 概要
生成し得る値の最小値を取得する。


## 戻り値
テンプレートパラメータ`c`が`0`であれば`1`、そうでなければ`0`を返す。


## 計算量
定数時間


## 例
```cpp example
#include <iostream>
#include <random>

int main()
{
  std::uint32_t min_value = std::minstd_rand::min();
  std::cout << min_value << std::endl;
}
```
