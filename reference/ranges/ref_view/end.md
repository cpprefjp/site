## 概要

番兵を取得する。


## 戻り値
メンバ変数`r_`として保持しているRangeへのポインタがあるとして、以下を返す：

```cpp
ranges::end(*r_);
```



## 例
```cpp example
#include <iostream>
#include <vector>
#include <ranges>

int main() {
  std::vector<int> v = {1, 2, 3};
  auto r = std::ranges::ref_view(v);
  auto it = r.begin();
  auto end = r.end();

  while (it != end) {
    std::cout << *it << ' ';
    ++it;
  }
}
```
