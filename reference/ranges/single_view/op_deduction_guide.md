## 概要

[`single_view`](../single_view.md)クラステンプレートの型推論補助。


## 例
```cpp example
#include <iostream>
#include <ranges>

int main() {
  auto r = std::ranges::single_view(1);
  for (int x : r) {
    std::cout << x << std::endl;
  }
}

```

### 出力
```
1
```
