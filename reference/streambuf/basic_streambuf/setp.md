## 概要
出力部分列の各ポインタを設定する。

## 事後条件
`pbeg ==` [`pbase()`](pbase.md) および `pbeg ==` [`pptr()`](pptr.md) および `pend ==` [`epptr()`](epptr.md)。

## 例
```cpp example
#include <iostream>
#include <streambuf>

// streambufを継承したクラス
class dummy_buf : public std::streambuf {
public:
    dummy_buf(char* begin, char* end) {
        // 出力列の領域を指定
        setp(begin, end);
    }
};

int main() {
    char space[10] = {};
    dummy_buf buf{space, space + 10};
    buf.sputc('A');  // 3文字書き込む
    buf.sputc('B');
    buf.sputc('C');
    std::cout << space << std::endl;  // 書き込んだ文字列を出力
}
```
