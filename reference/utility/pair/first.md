## 概要
[`pair`](/reference/utility/pair.md)クラスのメンバ変数で、1つめの要素を表す。

## 例
```cpp example
#include <utility>
#include <iostream>

int main() {
  std::pair<int, int> p(1, 2);
  std::cout << p.first << std::endl;
}
```
