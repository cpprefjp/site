```cpp
T min() const;
```

## 概要
最小の要素を取得する。


## 戻り値
`valarray`オブジェクトに含まれる最小の要素を返す。

最小要素の計算には、要素型`T`の`operator<`を使用する。計算の順番は未規定。

要素数が`0`の場合、未定義動作を引き起こす。要素数が`1`の場合、`0`番目の要素を返す。


## 例
```cpp example
#include <iostream>
#include <valarray>

int main()
{
  const std::valarray<int> va = {3, 1, 4};

  int result = va.min();
  std::cout << result << std::endl;
}
```
