# version
* version[meta header]
* cpp20[meta cpp]

`<version>`ヘッダでは、実装依でのC++標準ライブラリのバージョン情報を提供する。定義されるバージョン情報は実装ごとに決められるため、標準ではこのヘッダは空である。

## 機能テストマク

本ヘッダをインクルードすることにより、標準で提供される全てのライブラリ機能テストマクを利用することができる。

- [C++14 機能テストマク](/lang/cpp14/feature_test_macros.md)
- [C++17 機能テストマク](/lang/cpp17/feature_test_macros.md)
- [C++20 機能テストマク](/lang/cpp20/feature_test_macros.md.nolink)
